在 JSX 中， `<MyComponent />` 是合法的，而 `<MyComponent>` 就不合法。 所有的标签都必须闭合，可以是自闭和的形式，也可以是常规的闭合。

> 注意：
>
> 所有 React component 都可以采用自闭和的形式：`<div />`. `<div></div>` 它们是等价的。
